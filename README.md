Projekti nimi: Job finder

Osalejate nimed: Demjan Kipriyanov Vseslav Golenduhhin Aleksei Zassimovits

Projekti idee: Tööpakkumiste otsimine erinevatelt veebilehtedelt

Lühikirjeldus: Meie programm võimaldab otsida tööpakkumisi erinevatelt veebilehtedelt. Programm kuvab töö nimetuse, asukoha, lingi tööpakkumisele ning lisainfo.

Github https://github.com/projekt4267/-T-otsing
Kuvatav info:
Ametikoht
Ettevõte
Linn
Palk
Link tööpakkumisele

Programmi funktsioonid:
Tööpakkumiste otsing päringu järgi
Filtreerimine palga ja linna järgi
Tööpakkumise avamine topeltklõpsuga
Andmete salvestamine CSV ja TXT failina

Kuidas kasutada:
Sisesta ametikoht
Vajuta otsingu nuppu
Vajadusel kasuta filtreid
Topeltklõps — ava tööpakkumine

customtkinter, requests, pillow teegid, mida tuleb alla laadida
